# Taraji Schools Website & Admin Platform

This repository contains the official website and internal admin platform for **Taraji Afrika Education Group**, including Brookside International School campuses.  
The project is designed to deliver a modern, scalable, and content-managed digital presence for an international school group.

---

## Project Overview

Taraji Schools is a full-stack web application that includes:

- A public-facing school website (marketing & information)
- An admin dashboard for content and operations management
- Scalable backend architecture for future growth

The platform supports school branding, admissions communication, academic content, and internal administration.

---

## Schools Covered

- Brookside International School – Westlands Campus  
- Brookside International School – Redhill Campus  

(Architecture supports adding more campuses seamlessly.)

---

## Features

### Public Website
- Home, About, Academics, Campuses, Admissions, Contact pages
- Responsive, mobile-first design
- SEO-friendly structure
- Optimized performance and accessibility

### Admin Dashboard
- Secure authentication
- Manage blog posts and announcements (CRUD)
- Manage enquiries and contact submissions
- Track admissions leads
- Upload and manage images/media
- Scalable structure for payments and reports

### Communication
- Email notifications (admin & user)
- Contact and enquiry forms
- Extensible for newsletters and campaigns

---

## Tech Stack

### Frontend
- React + Vite
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- REST API architecture

### Services & Integrations
- Cloudinary – media storage
- Resend – transactional emails
- Paystack – payments (future-ready)
- JWT – authentication
